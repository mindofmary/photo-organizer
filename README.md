# Photo Organizer Pro

Aplicação Python para organizar automaticamente bibliotecas de fotos e vídeos.

## Funcionalidades

- Interface gráfica moderna
- Organização automática por mês
- Deduplicação
- Suporte HEIC, JPG, PNG
- Suporte MP4 e MOV
- Estatísticas de biblioteca

Photo Organizer is a comprehensive Python application that simplifies managing large photo and video collections through automatic organization, deduplication and insightful analytics. It supports various formats like HEIC, JPG, PNG, MP4 and MOV, making it versatile for diverse media libraries.

### Why Photo Organizer?

This project helps developers and media enthusiasts efficiently organize, analyze and maintain their media assets. The core features include:
* 🧩🖼️ **Media Deduplication**: Uses perceptual hashes and metadata to identify and eliminate duplicates, saving storage space.
*  🚀📁 **Format Support**: Handles multiple image and video formats seamlessly within a unified system.
*  🎯🖥️ **User-Friendly GUI**: Provides an intuitive interface with threading support for smooth, responsive interactions.
*  📊📈 **Statistical Insights**: Generates visual summaries and trends to inform media management decisions.
*  📦🛠️ **Standalone Packaging**: builds single executable files for easy deployment and distribution.
